Objective:

The objective of this project is to build a simple and interactive To-Do List web application that helps users manage their daily tasks.
This project focuses on practicing JavaScript logic, DOM manipulation, and UI styling while creating a useful real-world application.

Tools & Technologies Used:

HTML – For structuring the web page

CSS – For styling the interface and improving user experience

JavaScript – For handling task logic and interactivity

VS Code – Code editor

Web Browser – For running and testing the application

 Steps Performed:

Designed the basic structure of the to-do app using HTML

Created an input section to add new tasks

Implemented task handling logic using JavaScript

Added functionality to:

Add tasks

Mark tasks as completed

Move tasks between Pending and Completed sections

Edit existing tasks

Delete tasks

Displayed the date and time when tasks are added or completed

Enhanced the UI using advanced CSS styling, dark theme, and transitions

 Features:

Add new tasks easily

Separate Pending Tasks and Completed Tasks sections

Mark tasks as completed or undo them

Edit and delete tasks anytime

Displays date and time for better tracking

Modern and stylish user interface with smooth transitions

 Outcome:

The final outcome is a fully functional To-Do web application that allows users to efficiently organize and manage tasks.
This project strengthened my understanding of JavaScript functions, event handling, DOM updates, and CSS styling.
